# redline-ms-word-with-apple-automator-applescript

## Setup:
- Top of screen settings in Automator should be: Workflow receives current: files and folders in Finder.
- In Settings on Mac under Privacy & Security, you may need to give Automator "Full Disk Access"

## To Run:
- Select the two files in Finder, pick the before version first, right click on them, then Quick Actions > select script
- That will do the work for you of running a redline between the two files.
